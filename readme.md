# Focusly app

Aplicacion web para gestionar tareas por categorias

## Funcionalidades

- Registro e inicio de sesion 
- Crear categorias
- Agregar tareas
- Marcar tareas como completadas
- Colores por categoria
- Animaciones 

## Tecnoligias usadas

- HTML
- CSS
- JavaScript

## Estructura 

- /CSS => estilos
- /js => logica
- HTML => vistas

## Datos 

Se guardan en LocalStorage (sin backend)
